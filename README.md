# hyperspace

A screensaver or demoscene, whatever, that opens you up to a new space, hyperspace.
Made with [Macroquad](https://macroquad.rs) and [Rust](https://www.rust-lang.org).

![Example](example.gif)

## Compile from sources

```bash
cargo build
cargo run
```

## Customization

+ You can replace the star image by using the argument `-t TEXTURE_PATH`.
